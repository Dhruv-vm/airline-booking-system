# ✈️ Airline Booking System

An Airline Booking System developed as part of my Class 12 Computer Science Board Project. The project simulates the workflow of a real-world airline reservation platform by allowing users to search flights, book tickets, manage passenger information, and generate boarding passes.

This project marked one of my first large-scale software development experiences and introduced me to concepts such as database management, file handling, user authentication, and application design.

---

## 🚀 Features

- User Login and Authentication System
- Flight Search Functionality
- Ticket Booking System
- Passenger Information Management
- Automated Boarding Pass Generation
- Booking Record Management
- CSV-Based Data Storage
- MySQL Database Integration
- GUI-based interaction using Python libraries

## 📋 Functionalities

### 🔐 User Authentication
The project includes a dedicated login system that manages user authentication and access control before entering the booking platform.

### 🛫 Flight Booking
Users can browse available flights, select their preferred options, and complete the booking process.

### 👤 Passenger Management
The application stores passenger information and booking details for future reference and ticket generation.

### 🎫 Boarding Pass Generation
A boarding pass is automatically generated after a successful booking, providing a realistic airline reservation experience.

### 🗂 Data Management
CSV files are used to store and retrieve booking information, passenger details, and operational data while MySQL is utilized for login credentials and user management.
